# Présentation

boutique_pro est une application de gestion de stock simple qui permet :

- de gérer des catégories de produits,

- d’ajouter, modifier et lister des produits,

- de suivre automatiquement l’historique des mouvements de stock (entrées/sorties).

# Technologies utilisées

- Langage : Python 

- Base de données : MySQL

- Connecteur MySQL : mysql-connector-python

- Interface : Ligne de commande / terminal
